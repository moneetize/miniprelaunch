# Moneetize Merch Logo Exports

Generated formats:

- `logo_horizontal_[full_color|black|white]`
- `logo_stacked_[full_color|black|white]`
- `logo_message_[full_color|black|white]`
- `logo_icon_only_[full_color|black|white]`

Each format includes:

- SVG vector master
- Transparent PNG at 1024 x 1024
- Transparent PNG at 3000 x 3000

Merch usage:

- Small placements such as hats, pens, pins, and embroidery should use icon-only.
- Wide placements should use horizontal.
- Centered placements such as T-shirts, totes, and hero product previews should use stacked or message.
- Embroidery should use black or white single-color art and should omit the slogan.
- Full-color assets may use gradients; black and white outputs are high-contrast single-color options for light and dark backgrounds.
